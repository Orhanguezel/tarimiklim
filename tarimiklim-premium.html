II. Çekirdek Modüller

Üç servis. Bir altyapı.
Binlerce parsel için doğru karar.

Her modül bağımsız API olarak yayında; birlikte kullanıldığında bir günün içinde ekim, sulama ve hasat kararlarını veri temelli hale getirir.

01 — Risk
Don riski, dört faktörlü.
Sıcaklık düşüş hızı, nem, rüzgâr ve bulut örtüsünü harmanlayan bir skor. Meyve bahçeleri ve açık tarla için farklı eşikler; ≤-2°C'de SMS ve Telegram üzerinden çiftçiye gece uyarısı.
  • Skor hesaplama aralığı1 saat
  • İleri tahmin penceresi72 saat
  • Bildirim kanallarıSMS · TG · Push
  • Doğruluk (2024 sezonu)%97.2
02 — Tahmin
7 gün hava, saat çözünürlükte.
OpenWeatherMap birincil, Open-Meteo fallback; ilçe seviyesinde çözünürlük, parsel koordinatına düşen saatlik seri. Redis üzerinde 30dk TTL ile cache, BullMQ ile otomatik tazeleme.
  • Kaynak sayısı2 + yerel
  • Çözünürlükİlçe · 9 km
  • TazelemeHer 30 dk
  • Parametreler14 metrik
03 — Sulama
Sulama planı, evapotranspirasyonla.
ETo hesabı, yağış tahmini ve toprak türüne göre her parsele özel sulama takvimi. "Bu hafta Çarşamba gecesi sulama yapma" seviyesinde somut tavsiye; damla, yağmurlama veya salma için ayrı formüller.
  • Yöntem desteği3 tür
  • Ürün şablonu48 hazır
  • Su tasarrufu (pilot)%22
  • EntegrasyonIoT sensör

IV. API & Widget

Üç satırda ilk tahmininizi çekin.

REST endpoint'leri veya tek satırlık iframe widget — hangisi sizin ürününüz için daha kolaysa. Aynı altyapı Bereketfide ve VistaSeed sidebar'larında şu anda çalışıyor.

terminal · curl
# 7 günlük tahmin + don riski skoru
curl 'https://api.tarimiklim.com/v2/forecast' \
  -H 'Authorization: Bearer tk_live_...' \
  -d '{ "lat": 37.5731, "lon": 32.7734, "days": 7 }'

# → response (kısaltılmış)
{
  "location": "Konya · Çumra",
  "updated_at": "2026-04-17T14:32:18Z",
  "frost_risk_24h": 0.78,
  "frost_level": "critical",
  "forecast": [
    { "t": "06:00", "temp": -2.1, "hum": 89 },
    { "t": "09:00", "temp": 4.8, "hum": 72 },
    // ...
  ],
  "irrigation_advice": "skip_wednesday_night"
}

REST API, JS SDK, iframe widget.

API anahtarınızı alın, 60 saniye içinde ilk isteğinizi atın. Rate-limit 10k req/dk · yanıt p95 48ms · TR bölgesinde Frankfurt edge'i.
